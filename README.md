GermanGenie Landing Page

This repository is a landing page for GermanGenie, a german-learnig app with a focused on Grammar. This was created as part of the university course "Content Marketing + SEO/SEM Strategie".

Project Overview

The goal of this project was to build a marketing‑oriented landing page that supports the broader content marketing and SEO/SEM objectives of the GermanGenie concept. The page was developed with a focus on clarity, user experience, and search engine visibility, applying the principles and strategies learned in the course.

Alongside this code, a written report titled “GermanGenie - SEO Report & SEA Strategy” was created. The report contains detailed analysis, optimization strategy, keyword research, and recommendations, and is not included in this repository.

Features:
Static landing page structure in the public folder
Includes basic site assets (images, icons, etc.)
Built to support content marketing and SEO/SEM strategy
Designed to be clear, accessible, and aligned with SEO best practices

How It Was Built:
This project was created using:

HTML for structure
CSS for styling
Javascript for basic functionality and performance optimization
Assets (images, icons) to support page content

No backend or dynamic features were implemented as the focus was on content positioning and marketing optimization.


The live page can be found at: [https://germangenie-landingpage.vercel.app/]
